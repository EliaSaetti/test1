INTRODUZIONE

JavaScript è un linguaggio di scripting che viene interpretato direttamente dal browser, e che quindi, si può definire un linguaggio
lato-client.
Il codice può essere scritto sia nello stesso file con estensione html ma anche in un file esterno con estensione js.

Perchè si chiama Java Script?
Quando JavaScript è stato creato, inizialmente aveva un altro nome "LiveScript". Anche Java nativo era molto popolare, quindi si è deciso che il posizionamento di un nuovo linguaggio come "fratello minore" di java sarebbe stato d'aiuto. Man mano che si è evoluto, JavaScript è diventato un linguaggio completamente indipendente con una porpria specifica chiamata ECMAScript e non ha alcuna relazione con Java.

Cosa può fare JavaScript nel browser?
Il moderno JavaScript è un linguaggio di programmazione "sicuro". Non fornisce un accesso di basso livello alla memoria o alla CPU, perché inizialmente è stato creato per i browser che non lo richiedono. JavaScript nel browser può fare tutto ciò che riguarda la manipolazione della pagina web, l'interazione con l'utente e il server web.
Ad esempio, JavaScript nel browser è in grado di:
- Reagisci alle azioni dell'utente, esegui con clic del mouse, movimenti del puntatore, pressioni di tasti.
- Inviare richieste in rete a server remoti, scaricare e caricare file (tecnologie AJAX e COMET).
- Ottieni e imposta cookie, fai domande al visitatore, mostra messaggi.

Cosa rende JavaScript unico?
- Piena integrazione con HTML/CSS
- Supportato da tutti i principali browser e abilitato per impostazione predefinita
- Facile l'utilizzo

Le variabili
La maggior parte delle volte, un'applicazione JavaScript deve lavorare con le informazioni e per memorizzarle vengono utlizzate le variabili. Una variabile è una "memoria denominata" per i dati. Possiamo utilizzare le variabili per memorizzare gadget, visitatori e altri dati.
Per creare una variabile in JavaScript, utilizza la parola chiave. var

Ogni variabile non ha bisogno che le si venga specificata se String, int, char ecc. Nell'immagine si nota come qualsiasi tipo di dato possa essere affidato ad una variabile.

I cicli , detti anche strutture iterative, hanno una grande importanza in programmazione.
